# Linear combination of two vectors in C
  This guide is meant to give a detailed description of how the C code `vector_linearcombination` is built and works.

## Goal of the code
  This C program has the goal of computing the linear combination of two N-dimensional vector, according to the formula D = a · X + Y, where:
- **a** is a scalar,
- **X** and **Y** are **N**-dimensional vectors, with the same value for all entries (**Xi**, **Yi** fixed for all i),
- **D** is the resulting vector.

## Compiling the code
To compile the code and obtain the executable file, use the `gcc` compiler in the following way:

```bash
gcc vector_linearcombination.c -o vector_linearcombination
```
The `-o` option allows to choose the name of the executable file.

## Structure of the code
- The user is asked to provide the constant values of **a**, **Xi**, **Yi** and the dimensionality **N** of the vectors from the command line, as shown in the following:
  ```bash
  ./vector_linearcombination a Xi Yi N
  ```
- If the syntax is not respected, an error will be printed out, saying `Usage: ./vector_linearcombination a Xi Yi N`.
- Some instructions will be printed out once the program is started.
- The **X** and **Y** vectors are built using dynamical allocation of memory, through pointers, as `float *X = (float*) malloc(N * sizeof(float));`.
- A function named `VectorLinearCombination(a, X, Y, N)` is then called, taking as input the scalar, the two initial vectors and the dimension N and build the linear combination vector **D** again dynamically allocating the memory.
- A check that the program successfully computed the result vector is implemented: the expected result for every entry is computed as `expected = a · Xi + Yi` and then compared to the resulting vector **D** entry values. If component `k` is different from the expected outcome, an error message will be printed out: `Component number k is wrong`.
- Finally, the dynamically allocated memory is freed through the command `free`.
